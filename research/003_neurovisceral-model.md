# 论文 003：神经内脏整合模型

**来源**：Schmalenberger 2019 引言中的理论框架

## 核心命题

心脏迷走神经控制（CVA/HRV）不是随机波动，而是中枢神经系统功能的一个"窗口"。

### 因果链

```
前额叶皮层 / 前扣带回 / 岛叶
       │
中央自主网络 (CAN)   ← ← 雌激素(E2)/孕酮(P4)受体密集区
       │
心脏迷走神经控制 (CVA / HRV)
       │
   ┌───┴───┐
   ↓       ↓
心理健康    身体健康
(情绪/认知)  (炎症/代谢/免疫)
```

## 个体间 vs 个体内变异

### 个体间（人与人不同）— 固定层
| 来源 | 隐链中如何处理 |
|---|---|
| 生物性别 | 当前聚焦女性数据 |
| 基因（COMT） | 可选长期 |
| 肥胖/药物/酒精/运动 | 生活方式协变量 |
| 工作压力 | 情绪日志标注 |

### 个体内（同一人不同时间）— 隐链的核心战场
| 来源 | 策略 |
|---|---|
| 昼夜节律 | 时间窗口标准化 |
| 身体运动 | 过滤运动时段 |
| 精神压力 | 事件标签 + delta 分析 |
| **月经周期（d=-1.32）** | **周期分层基线 ← 最高优先级** |
