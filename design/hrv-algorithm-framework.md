## 第 1 层：分轨处理

### 静息轨
- 晨起窗口：起床后 5 分钟内（未进食、未活动）
- 睡前窗口：入睡前最后 10 分钟
- 输出：静息 RMSSD、SDNN、HF
- 日间消耗指数 = 晨起静息 HRV - 睡前静息 HRV

### 事件轨
- 压力事件 delta：事件后 3 分钟内 - 事件前 5 分钟
- 恢复速率：从峰值偏移到恢复至基线所需时间
  - 快速恢复 (< 5min) → 高迷走神经弹性
  - 慢速恢复 (> 30min) → 自主神经调节下降

## 第 2 层：周期校准

### 阶段判定规则（基于 28 天标准周期）
```python
def get_cycle_phase(day_of_cycle):
    if 1 <= day <= 5:   return "menstrual"
    if 6 <= day <= 14:  return "follicular"   # 基线推荐窗口
    if 15 <= day <= 17: return "ovulatory"
    if 18 <= day <= 24: return "luteal"
    if 25 <= day <= 28: return "premenstrual" # CVA 最低窗口
```

### 归一化方法
- 每个阶段独立计算 mean_phase 和 std_phase
- z = (hrv_raw - mean_phase) / std_phase
- z > 0 → 高于该阶段预期水平
- z < 0 → 低于该阶段预期水平

## 第 3 层：Causal AI 三元验证

| HRV 变化 | 情绪标签 | 判定结果 |
|---|---|---|
| ↑ | 积极 | 调节成功 |
| ↑ | 消极 | 调节努力 |
| ↓ | 积极 | 可能是基线波动 |
| ↓ | 消极 | 调节失败 / 压力超载 |

## 第 4 层：中医映射

| 西医指标组合 | 中医证型 | 判断逻辑 |
|---|---|---|
| 静息 HRV 持续偏低 + 恢复慢 | 气血不足 / 脾虚 | 基线储备低，恢复能力差 |
| 事件后 HRV 剧烈波动 + 恢复慢 | 肝郁气滞 | 压力反应过度，疏泄功能弱 |
| 周期校准后 HRV 仍异常 | 痰气互结 | 排除周期影响后仍偏离 |
| 静息 HRV 正常但事件反应不足 | 阳气不足 | 基线可但应战能力弱 |

## 第 5 层：输出

### 每日调节指数（0-100）
- 0-30：🔴 需要关注
- 31-60：🟡 注意调节
- 61-80：🟢 状态良好
- 81-100：🟣 最佳状态

### 周期报告
- 每周期末自动生成
- 包含：该周期调节能力趋势、异常事件标记、跨周期对比
